# 🗺️ U.S States Game

An interactive quiz game built using Python and Turtle graphics where you guess all 50 U.S states.

---

## 🎮 Features

* Interactive U.S map
* Score tracking
* Saves missed states to a CSV file
* Clean and simple UI

---

## 🧠 How It Works

* Type the name of a U.S state
* If correct → it appears on the map
* Type **Exit** to quit and save missing states
